Oque é commit? oque é push? oque é pull???

para isso agente tem que seber de algumas coisas...

O commit é da pessoa a o repositorio.

e o push é do repositorio para o repositorio remoto

e o pull e do repositorio remoto para o repositorio

Nas aulas daqui o Git é o repositorio local e o Github é o Repositorio remoto.

Agora você sabe fazer um commit um push e um pull!!!
